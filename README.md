BXtexlogo Package
=================

LaTeX: Additional TeX-family logos

Blah....

### System requirement

  * TeX format: LaTeX.
  * TeX engine: Anything.
  * Dependent packages:
      - hologo
      - cjhebrew (if `\logoAleph` and `\logoLamed` are used)

### Installation

  - `*.sty` → $TEXMF/tex/latex/BXtexlogo

The bxtexlogo Package
---------------------

### Package Loading

    \usepackage[<option>,...]{bxtexlogo}

All the options are passed to hologo.

### Usage

You can do `\BibTeX`.

You can do `\e-pTeX`.

You can do `\upLaTeX`.

You can do `\JBibTeX`.
